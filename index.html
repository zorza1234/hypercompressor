Frequently Asked Questions

How does client-side compression work?

Hyper Compressor utilizes the modern HTML5 Canvas API built directly into your web browser. When you drop an image, your browser analyzes and redraws the image at a smaller file size locally, without ever needing to communicate with an external server.

Which image formats are supported?

Currently, we support the most common web formats: JPG (JPEG), PNG, and WebP. Compressing a PNG usually converts it to a high-efficiency JPEG or WebP internally to achieve maximum space savings.
